Sourcecode for BDB JE is available online from "www.oracle.com":http://www.oracle.com/technetwork/database/berkeleydb/downloads/index.html

This project builds with sbt 0.7.4. Run @sbt update compile@ to compile. Run @sbt run@ to run the tests (choose test number from the sbt menu).
